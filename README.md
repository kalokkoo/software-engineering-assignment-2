# 簡介
這個是一個簡單的記賬功能表，簡單的輸入（事項，金額等）程式會自動的把數據轉換成圓餅圖，讓我們的數據更加的視覺化
## 檔案講解
```
expenses.csv：是一個資料庫所有的輸入的資料數據之後都會記錄在裏面
input.py：也就是我們的輸入檔
output.py：也就是我們的輸出檔
.vscode：這是一個基於 Python 的跨平台圖形化記帳系統，結合了 tkinter 視窗介面與 matplotlib 數據分析圖表，並具備自動修正中文亂碼與格式化金額的實用功能。
```
## 環境設定
### 安裝套件
```
pip install matplotlib
pip install tkinter
pip install platform
```
## 執行説明
直接執行output.py 并輸入數據即可
